Overview

The Bank Statement Import feature automatically processes your bank statement files and matches transactions to customers and suppliers in Opera, eliminating manual data entry.

Key Features

  • Multi-format support: CSV, OFX, QIF, MT940
  • Auto-detection of bank account from file
  • Fuzzy matching to customers and suppliers
  • Duplicate detection prevents double-posting
  • Support for receipts, payments, and refunds
  • Import from file OR email attachments

Match Sources

Transactions are matched using: Aliases (learned from previous imports), Fuzzy matching (name similarity), and Enhanced matching (phonetic + n-gram algorithms).

Learning System

When you manually assign an account, the system saves this as an alias. Next time "UNKNOWN SENDER REF123" appears, it will automatically match to U001.
